# 🧠 AutoMind — AI-ассистент для автовладельцев

## Быстрый старт

### Требования
- Node.js 18+ (скачать: https://nodejs.org)
- npm (идёт вместе с Node.js)

### Установка и запуск

```bash
# 1. Распакуй архив и перейди в папку
cd automind

# 2. Установи зависимости
npm install

# 3. Запусти dev-сервер
npm run dev
```

Открой в браузере: **http://localhost:3000**

> 💡 Для мобильного вида — открой DevTools → Toggle Device Toolbar (Ctrl+Shift+M)

### Для теста на телефоне в локальной сети

```bash
# Узнай свой IP
# macOS: ifconfig | grep inet
# Windows: ipconfig

# Запусти с привязкой к IP
npx next dev -H 0.0.0.0
```

Затем на телефоне открой: `http://ТВОЙ_IP:3000`
